# InterVerse API

Backend cloud service for the InterVerseSG immersive campus platform.

## Initial scope

- Health endpoint for cloud deployment
- Secure OpenAI integration through server-side environment variables
- Structured commands for Unreal Engine and Meta Quest
- Builder command validation
- Automated tests and GitHub Actions

## Planned deployment

The service is designed for Render using `render.yaml` and exposes `/healthz` for health checks.
